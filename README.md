## Describe
This is the teaching repository of the Operations Department of the Yanshan University Student Net Information Association.

## 关于本仓库
本仓库主要用以存放第九届运维部授课所用教案及工具等，以便于同学们日后随时复习。（授课部长会将授课所用教案及工具放在一个目录下，并附上md说明文档）
